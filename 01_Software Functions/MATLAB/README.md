재사용 가능한 MATLAB 함수·스크립트를 모읍니다.

## 올리는 법

파일 맨 위에 5줄 헤더 주석을 붙입니다 ([`../../_templates/HEADER.md`](../../_templates/HEADER.md)):

```matlab
% 이름      : myFunction
% 용도      : (한 줄 설명)
% 작성자    : 홍길동 / 2026
% 사용법    : y = myFunction(x, opt)
% 의존성    : Signal Processing Toolbox / 없음
```

- 파일명은 함수명과 일치, 공백 없이 (예: `bandpass_filter.m`).
- 여러 파일로 된 항목은 항목 폴더로 묶고 `README.md`를 둡니다.
