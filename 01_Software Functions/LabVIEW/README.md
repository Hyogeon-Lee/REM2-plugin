# LabVIEW

재사용 가능한 LabVIEW VI·코드를 모읍니다.

## 올리는 법

VI는 파일 안에 헤더 주석을 넣기 어려우므로 다음 중 하나로 설명을 답니다:

- **VI Properties → Documentation → VI Description** 칸에 5줄 헤더(이름·용도·작성자·사용법·의존성)를 적습니다.
- 또는 같은 이름의 `<VI이름>.md` 파일을 함께 두고 [`../../_templates/HEADER.md`](../../_templates/HEADER.md)의 Markdown 양식으로 채웁니다.

- 라이브러리·프로젝트 단위는 폴더로 묶고 `README.md`를 둡니다.
- LabVIEW 버전을 헤더 의존성 칸에 적어 주세요 (예: `LabVIEW 2023 Q3`).
