# 01_Software Functions

연구·개발에 재사용 가능한 **코드, 함수, 스크립트**를 언어별로 모읍니다.

## 하위 분류

| 폴더 | 내용 |
|---|---|
| [`MATLAB`](MATLAB) | MATLAB 함수·스크립트 |
| [`LabVIEW`](LabVIEW) | LabVIEW VI·코드 |

다른 언어는 같은 위치에 언어명 폴더(예: `Python`, `C`)를 새로 만들어 사용합니다.

## 올리는 법

- **단일 함수/스크립트**: 해당 언어 폴더에 파일을 넣고 맨 위에 5줄 헤더 주석을 붙입니다.
- **여러 파일로 된 항목**: 항목 이름 폴더로 묶고 안에 `README.md`를 둡니다.

규칙은 [`../CONTRIBUTING.md`](../CONTRIBUTING.md), 헤더 양식은 [`../_templates/HEADER.md`](../_templates/HEADER.md) 참고.
