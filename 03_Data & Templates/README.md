# 03_Data & Templates

**예제 데이터셋, 보고서·문서 템플릿, 매뉴얼** 등 코드도 참고링크도 아닌 공유 자료를 모읍니다.

## 담는 내용

- 예제·샘플 데이터셋 (테스트·튜토리얼용)
- 보고서·논문·발표 문서 템플릿
- 장비 매뉴얼, 작업 절차서(SOP)

## 올리는 법

- 데이터·템플릿 파일은 헤더를 넣을 수 없으므로, **같은 폴더에 설명용 `.md`** 또는 폴더 `README.md`를 두어 이름·용도·작성자·출처를 적습니다.
- **대용량 파일 주의**: 대략 100MB를 넘는 데이터는 직접 올리지 말고, 보관 위치(공유드라이브 등) 링크를 `.md`에 적습니다.
- 미공개 연구데이터·개인정보가 포함된 데이터는 올리지 않습니다 ([`../CONTRIBUTING.md`](../CONTRIBUTING.md) 참고).
