# 00_AI Workflow

AI를 연구·개발 워크플로우에 활용하기 위한 **스킬, 에이전트 구성, 프롬프트, MCP 설정** 등을 모읍니다.

## 담는 내용

- Claude/LLM용 스킬 (`SKILL.md` + 스크립트 묶음)
- 에이전트 팀·멀티에이전트 워크플로우 구성
- 재사용 프롬프트, MCP 서버 설정, 연구 자동화 흐름

## 올리는 법

폴더 단위 항목은 항목 이름의 하위 폴더로 묶고, 그 안에 `README.md`(또는 `SKILL.md`)를 둡니다. 규칙은 [`../CONTRIBUTING.md`](../CONTRIBUTING.md), 양식은 [`../_templates`](../_templates) 참고.

## 등록된 항목

| 항목 | 용도 | 위치 |
|---|---|---|
